# Orientation Step 1: Welcome

## Purpose
Introduce what Trike is and how it helps complete beginners build with AI.

## Key Messages

**What is Trike:**
- Training wheels for building with AI
- You build real projects (not tutorials)
- Claude Code does the coding, you guide and verify
- Learn what you need, when you need it
- Understand everything you build (never copy-paste)

**The Training Wheels Metaphor:**
- Enables what you couldn't do alone
- Teaches you gradually
- Comes off when you're ready

## Tone
- Welcoming and encouraging
- Brief and clear (fits in ~20 lines)
- Exciting without overwhelming

## Navigation
After welcome, user can:
- `/trike:next` - Continue orientation (recommended)
- `/trike:plan` - Skip to planning (if familiar with Claude Code)
- `/trike:help` - See all commands
